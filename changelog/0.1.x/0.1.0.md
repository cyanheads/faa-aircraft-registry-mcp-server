# 0.1.0 — 2026-06-13

## Added

- **`faa_lookup_registration`** — decode one N-number to its full pre-joined active record: make/model, engine, year, airworthiness, registration status, Mode S (ICAO 24-bit) code, and registered owner. Accepts `N12345` or `12345`.
- **`faa_get_registration_status`** — resolve an N-number across all three status files (active MASTER, deregistered DEREG, reserved RESERVED) in priority order, returning a discriminated `recordType`; a never-issued number resolves to `unknown` rather than an error.
- **`faa_search_registrations`** — FTS5 search of active registrations by owner name, make/model, state, aircraft type, or Mode S code. Discloses truncation at the limit.
- **`faa_search_aircraft_types`** — FTS5 search of the aircraft reference table by manufacturer/model name, aircraft type, or category to discover manufacturer-model codes.
- **`faa_get_aircraft_type`** — decode a 7-char manufacturer/model/series code to aircraft specs (category, type, engine, seats, weight class, cruise speed, type-certificate data).
- **`faa://registration/{nNumber}`** — read-once resource mirroring the `faa_lookup_registration` payload.
- **`registry-service`** — embedded SQLite + FTS5 mirror (framework MirrorService) built from the FAA Releasable Aircraft Database, pre-joining MASTER → ACFTREF → ENGINE and decoding coded fields at query time (raw code + decoded label surfaced for every coded field).
- **Mirror lifecycle** — `mirror:init` / `mirror:refresh` / `mirror:verify` CLI scripts (full out-of-band build; the HTTP server schedules a daily refresh aligned to the FAA's nightly re-release). Ingester enumerates the nine `MASTER-1.txt`…`MASTER-9.txt` parts.
- **`FAA_REDACT_OWNER_PII`** — fail-safe owner-PII redaction, redacted by default. When redacted, owner name/address are dropped from output AND owner-name search is disabled; unset/empty/malformed resolves to redacted.

## Security

- Owner-PII redaction defaults to on and fails safe — a deployment can never leak registrant names/addresses by omission or misconfiguration. Both DEREG address blocks (mailing and physical) are scrubbed, and owner-name search is disabled under redaction so the search input cannot be used to confirm a person↔aircraft link.
