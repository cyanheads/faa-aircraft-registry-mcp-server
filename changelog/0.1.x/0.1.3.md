# 0.1.3 — 2026-06-20

## Changed

- **Framework** — `@cyanheads/mcp-ts-core` `^0.10.6` → `^0.10.9`. Picks up the fresh-scaffold `devcheck` git guards (changelog-sync, TODOs/Secrets/Antipatterns skip cleanly without a `.git`), the new `check-dependency-specifiers` step (rejects floating `latest`/`*`/dist-tag specifiers in `package.json` + the `bun.lock` workspaces map), plugin-marketplace manifest validation in `lint:packaging`, the `DuckdbProvider.describe()` ambiguous-column binder fix, Canvas SQL `invalid_sql` prepare-failure classification, the `ctx.content` media-block collector, and the `OTEL_SERVICE_NAME` seeding doc correction.
- **`devcheck.config.json`** — added `packaging.pluginManifests: true`, enabling the new `lint:packaging` checks for `.claude-plugin/plugin.json` and `.codex-plugin/*.json` (non-empty descriptions; unscoped display identity; full scoped `npx -y` install arg).
- **Agent skills** — re-synced 14 `skills/<name>/SKILL.md` to the framework's 0.10.9 versions (`add-tool`, `api-auth`, `api-canvas`, `api-config`, `api-context`, `api-errors`, `api-services`, `api-telemetry`, `field-test`, `git-wrapup`, `orchestrations`, `polish-docs-meta`, `report-issue-local`, `tool-defs-analysis`).
- **devcheck scripts** — re-synced 6 framework scripts (`build-changelog`, `check-framework-antipatterns`, `check-skill-versions`, `devcheck`, `lint-packaging`, and the new `check-dependency-specifiers`).
